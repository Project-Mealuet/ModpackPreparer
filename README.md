# MealuetServerpackPreparer
简化Mealuet Server开服流程

# 功能  
- 服务端
  - [x] 填写`server.properties`；
  - [x] 同意`eula.txt`；
  - [x] 下载`authlib-injector.jar`；
  - [ ] 下载**Yes Steve Model**；
  - [ ] 添加YSM模型文件。
- 客户端
  - [x] 下载**Custom Skin Loader**；
  - [x] 将`MealuetSkin`添加到皮肤加载列表中；
  - [x] 下载**i18n**；
  - [x] 添加服务器列表；
  - [x] 配置游戏语言；
  - [ ] 下载**Yes Steve Model**。

# 使用  
⚠ 只在Windows系统上进行了测试  
1. 下载最新的Release；
2. 直接运行，按照指引输入参数。
